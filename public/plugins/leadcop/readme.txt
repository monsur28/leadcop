Advanced server-side email validation and typo correction for all major WordPress forms.

== Description ==

LeadCop protects your WordPress forms from disposable emails, temporary emails, role accounts, and invalid domains. 
It integrates seamlessly with Contact Form 7, WPForms, Gravity Forms, WooCommerce, Ninja Forms, and Elementor Pro.

The LeadCop plugin acts as a lightweight, secure bridge to the LeadCop API, ensuring your backend validation rules are strictly enforced without slowing down your WordPress site.

= Features =
* Instantly block disposable and temporary email addresses.
* Restrict role-based accounts (e.g., admin@, support@).
* Enforce custom domain blocklists.
* Works automatically with 6 major form plugins.
* Zero configuration required beyond connecting your API key.

== Installation ==

1. Upload the `leadcop` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the new 'LeadCop' menu in your WordPress admin.
4. Enter your LeadCop API Key to activate protection.

== Frequently Asked Questions ==

= Do I need a LeadCop account? =
Yes, you need an active LeadCop account and API key to use this plugin.

= Which form plugins are supported? =
We currently support Contact Form 7, WPForms, Gravity Forms, WooCommerce checkout, Ninja Forms, and Elementor Pro forms.
